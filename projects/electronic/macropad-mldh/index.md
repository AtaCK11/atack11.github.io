This project is an experimental macropad built to evaluate the **RP2350** microcontroller for a larger, more complex input device.
The goal is to explore hardware capability, firmware workflow, early.


- **RP2350 board**
- **128×32 display**
- **9 keyboard switches**
- **2 rotary encoders**
